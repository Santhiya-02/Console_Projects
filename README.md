# Console Projects (Java)

This repository contains multiple Java console-based applications.

## Projects

### 1. ATM System
- User & Admin login
- Deposit, Withdraw, Balance check
- Change PIN
- Account Transfer
- Mini Statement

## Technologies
- Java
- OOP Concepts

## How to Run
1. Navigate to project folder:
   cd ATM-System

2. Compile:
   javac *.java

3. Run:
   java ATM
